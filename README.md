# TaskMaster

Bem‑vindo ao TaskMaster — sistema de **gestão de tarefas colaborativas**, projetado para equipes distribuídas.

## Visão  
Permitir que equipes criem, organizem e acompanhem tarefas de forma simples, focada e colaborativa.
